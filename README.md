# ISPE Project | ISPE项目

ISPE is an experimental game project of space battle. ISPE是一款实验性太空作战游戏。

Please contribute by creating pull request. 请通过创建PR来贡献内容。

### Project Structure | 项目结构

#### Meta | 元数据

- README.md - 介绍文件

- LICENSE - GPLv3 许可证

- progamStructure.txt - 技术性的注释

#### Single files | 独立文件

- menu.tscn - 开始菜单的场景文件

- menu_page.gd - 开始菜单页面的对应脚本

- storyboard.tscn - 主场景文件

- overworld.gd - 主场景的对应脚本

#### Libs & Classes | 库和类文件

- orbitArgumentClassFiles - 存放轨道的内部参数的类的目录

- orbitDisplayClassFiles - 存放轨道的显示类的目录

- dataManagementClasses - 存放读取载具模板数据的目录

- calculateClassFiles - 存放轨道计算有关的函数的目录

- displayClassFiles - 存放各类显示类的目录

- spaceObjectClassFiles - 存放集中载具数据的类的目录

- UIClassFiles - 存放UI界面有关类的目录

- vehicleDataClassFiles - 存放载具数据和载具模板的类的目录
